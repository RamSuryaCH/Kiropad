## 2026-06-19 - Keyboard Accessibility Focus State
**Learning:** Adding a basic CSS `:focus-visible` outline to common interactive components like buttons is a minimal, non-intrusive way to dramatically improve keyboard navigation for users without affecting mouse-user aesthetics.
**Action:** Use `:focus-visible` coupled with `outline` and `outline-offset` instead of `:focus` or `border` overrides to maintain consistent component sizing and ensure accessibility rings only appear when necessary.
